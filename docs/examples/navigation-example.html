Sistema de Navegação Avançado

Este é um exemplo de como o sistema de navegação funciona na documentação do Dashboard Sabrina Costa.

Breadcrumbs

Os breadcrumbs mostram a localização atual na documentação, facilitando a navegação hierárquica.

Table of Contents (TOC)

A tabela de conteúdos é gerada automaticamente a partir dos cabeçalhos da página e oferece navegação rápida.

Funcionalidades do TOC

  • Geração automática a partir dos cabeçalhos
  • Comportamento sticky (grudado) em telas maiores
  • Destaque da seção ativa durante o scroll
  • Navegação suave entre seções
  • Suporte a teclado (ESC para fechar)

Navegação Anterior/Próxima

Links para navegar entre páginas relacionadas, melhorando o fluxo de leitura.

Botão Voltar ao Topo

Botão flutuante que aparece após rolar a página, permitindo retornar rapidamente ao início.

Design Responsivo

O sistema se adapta a diferentes tamanhos de tela, otimizando a experiência em dispositivos móveis.

Otimizações Mobile

  • TOC não fica sticky em telas pequenas
  • Breadcrumbs se reorganizam verticalmente
  • Botões de navegação se ajustam ao espaço disponível

Acessibilidade

O sistema inclui várias funcionalidades de acessibilidade para garantir que todos os usuários possam navegar eficientemente.

Navegação por Teclado

  • Suporte completo a navegação por teclado
  • Indicadores visuais de foco
  • Atalhos de teclado para funcionalidades principais

Modo Escuro

Suporte automático ao modo escuro baseado nas preferências do sistema.

Performance

O sistema foi otimizado para performance, com debounce no scroll e lazy loading quando necessário.

Técnicas de Otimização

  • Debounce no evento de scroll
  • Throttling de eventos de redimensionamento
  • Lazy loading de componentes pesados
  • CSS otimizado para animações suaves

Personalização

O sistema pode ser facilmente personalizado através de CSS e JavaScript.

Variáveis CSS

Use variáveis CSS para personalizar cores, espaçamentos e outros aspectos visuais.

Integração

Fácil integração em qualquer página HTML através de CSS e JavaScript.

Passos de Configuração

  1. Incluir o CSS de navegação
  2. Incluir o JavaScript de navegação
  3. Adicionar containers HTML necessários
  4. Configurar dados de navegação (opcional)

Conclusão

Este sistema de navegação oferece uma experiência completa e profissional para a documentação do Dashboard Sabrina Costa.
