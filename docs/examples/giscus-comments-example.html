Sistema de Comentários Giscus

Este é um exemplo de como o sistema de comentários Giscus funciona na documentação do Dashboard Sabrina Costa.

Funcionalidades

O sistema de comentários oferece:

  • Integração com GitHub: Usa GitHub Discussions para comentários
  • Autenticação automática: Login com conta GitHub
  • Identificação de página: Cada página tem sua própria discussão
  • Reações: Sistema de likes e reações
  • Moderação: Controle via GitHub Discussions

Configuração

Para configurar o sistema:

  1. Crie um repositório no GitHub
  2. Habilite GitHub Discussions
  3. Configure o Giscus no repositório
  4. Adicione o script à sua página

Personalização

O sistema pode ser personalizado através de:

  • Temas: Claro, escuro ou automático
  • Idioma: Português, inglês, etc.
  • Posição: Topo ou rodapé da página
  • Reações: Habilitar/desabilitar

Analytics

O sistema rastreia:

  • Carregamento dos comentários
  • Navegação entre discussões
  • Interações dos usuários

Responsividade

O sistema se adapta a diferentes tamanhos de tela:

  • Desktop: Layout completo
  • Tablet: Layout otimizado
  • Mobile: Layout compacto

Acessibilidade

Recursos de acessibilidade incluídos:

  • Navegação por teclado
  • Indicadores visuais de foco
  • Contraste adequado
  • Suporte a leitores de tela

Modo Escuro

Suporte automático ao modo escuro baseado nas preferências do sistema.

Integração

Para integrar o sistema:

  1. Incluir o CSS de comentários
  2. Incluir o JavaScript do sistema
  3. Adicionar container HTML
  4. Configurar repositório GitHub

Conclusão

O sistema de comentários Giscus oferece uma solução completa e integrada para discussões na documentação, aproveitando a infraestrutura do GitHub.
